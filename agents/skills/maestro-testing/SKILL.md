## Tools

- Use the `maestro` CLI to run tests.

## Running tests

- When asked to run tests, don't explore the codebase any more than is necessary -- just run the tests.

## Guidelines

- Only use local devices, never maestro cloud devices.

## Documentation

- General documentation is available at `https://docs.maestro.dev`.
- Documentation for the `maestro` CLI command is at `https://docs.maestro.dev/maestro-cli/maestro-cli-commands-and-options`.
